# セキュリティとプライバシーポリシー

## 概要

このアプリケーション（AI活用アイデアシート）は、**入力されたデータを一切保存しません**。すべての処理はブラウザ内で完結し、機密情報を含む業務課題を安全に入力できます。

## データの取り扱い

### データは保存されません

- ✅ **ローカルストレージ（localStorage）**: 使用していません
- ✅ **セッションストレージ（sessionStorage）**: 使用していません
- ✅ **Cookie**: 使用していません
- ✅ **IndexedDB**: 使用していません
- ✅ **サーバーへのデータ送信**: 行っていません
- ✅ **データベース**: 使用していません

### データの管理方法

すべてのデータはReactの`useState`フックで管理されており、ブラウザのメモリ（RAM）にのみ一時的に存在します。

```typescript
// app/page.tsx
const [ideas, setIdeas] = useState<AIIdea[]>([...]);
```

- データはページが開いている間のみ存在します
- ページを閉じると、すべてのデータは完全に消去されます
- ブラウザを再起動すると、データは消去されます

## 技術的な証明

### 1. クライアントサイドのみで動作

このアプリケーションはNext.jsの静的サイト生成（SSG）を使用しており、サーバーサイドの処理は一切ありません。

- **APIルート**: `app/api/`ディレクトリは存在しません
- **サーバーコンポーネント**: データベース接続や外部API呼び出しはありません
- **ビルド時生成**: すべてのページはビルド時に静的HTMLとして生成されます

### 2. エクスポート機能

エクスポート機能もブラウザ内で完結します。

```typescript
// components/ExportSection.tsx
const handleExport = () => {
  const content = getContent();
  const blob = new Blob([content], { type: 'text/plain;charset=utf-8' });
  // ファイルはローカルにダウンロードされるのみ
  // サーバーにアップロードされることはありません
};
```

- ファイルはお使いのデバイスにのみダウンロードされます
- サーバーにアップロードされることはありません
- クラウドストレージに保存されることはありません

### 3. 外部サービスとの連携

- **分析ツール**: Google Analytics、その他のトラッキングツールは使用していません
- **外部API**: データを送信する外部APIは使用していません
- **サードパーティサービス**: データを共有するサードパーティサービスは使用していません

## 検証方法

### ブラウザの開発者ツールで確認

1. **F12キー**で開発者ツールを開く
2. **Applicationタブ** > **Storage**を確認
   - Local Storage: 空であることを確認
   - Session Storage: 空であることを確認
   - Cookies: 空であることを確認
3. **Networkタブ**を開く
   - データを入力・エクスポートしても、サーバーへのリクエストがないことを確認

### ソースコードで確認

[GitHubリポジトリ](https://github.com/Michey0495/ai-idea-worksheet)で以下の点を確認できます：

1. **データ保存のコードがない**
   - `localStorage.setItem()`: 検索結果なし
   - `sessionStorage.setItem()`: 検索結果なし
   - `document.cookie`: 検索結果なし

2. **API呼び出しのコードがない**
   - `fetch()`: データ送信用のコードなし（エクスポート時のファイルダウンロードのみ）
   - `axios`: 使用していません
   - APIルート: `app/api/`ディレクトリが存在しない

3. **データベース接続のコードがない**
   - データベース接続ライブラリ: 使用していません
   - ORM: 使用していません

## 機密情報の安全な使用

このアプリケーションは、以下のような機密情報を含む業務課題を安全に入力できます：

- プロジェクト名
- システム名
- 顧客情報
- 業務フロー
- 要件定義の内容
- 設計書の内容

**注意事項**:
- エクスポートしたファイルは、お使いのデバイスに保存されます
- ファイルの管理にはご注意ください
- 不要になったファイルは削除してください

## プライバシー

### 収集する情報

このアプリケーションは、**一切の情報を収集しません**。

- 個人情報: 収集しません
- 使用状況: 収集しません
- 入力内容: 収集しません
- デバイス情報: 収集しません

### トラッキング

- Google Analytics: 使用していません
- その他のトラッキングツール: 使用していません
- 広告配信: 行っていません

## セキュリティのベストプラクティス

このアプリケーションを使用する際の推奨事項：

1. **HTTPS接続**: 常にHTTPSでアクセスしてください
2. **ファイル管理**: エクスポートしたファイルは適切に管理してください
3. **共有デバイス**: 共有デバイスで使用する場合は、使用後にページを閉じてください
4. **ブラウザの履歴**: 必要に応じてブラウザの履歴をクリアしてください
